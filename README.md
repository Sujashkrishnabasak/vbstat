# vbstat
Handouts and Notes BSc Statistics VisvaBharati

## Installation
```
git clone https://github.com/mrinalcs/vbstat.git
cd vbstat
code .
bundle
bundle exec jekyll serve
```

### Deploy to github pages
```
bundle lock --add-platform x86_64-linux
```
